*Note: This repository has been archived by the owner. It is now read-only.*

## Build your owns

### Clone from git

```shell
git clone https://github.com/Garfield550/izumis.moe-home.git

cd ./izumis.moe-home
```

### Install dependencies

```shell
yarn install
```

### Build site for production

```shell
yarn build
```
